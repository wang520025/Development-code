Git环境搭建 一，Git下载安装 1，下载git安装包，选择默认配置。 2，安装完后，右键会出现`git`菜单，选中 `Git Bash Here`，弹出命令窗口。 3，输入命令 `git --version` 查看`git`版本。 二，Git 使用前配置 1，配置提交人姓名：`git config --global user.name 提交人姓名`。 2，配置提交人姓名：`git config --global user.email 提交人邮箱`。 3，查看git配置信息：`git config --list` 。 三，Git使用 1）提交文件到Git仓库 1. git init` 初始化git仓库 2. `git status` 查看文件状态 3. `git add 文件列表` 追踪文件 4. `git commit -m 提交信息` 向仓库中提交代码 5. `git log` 查看提交记录 2）撤销 1. git checkout 文件 用暂存区中的文件覆盖工作目录中的文件 2. git rm --cached 文件名 将文件从暂存区中删除 3. git reset --hard 提交ID 恢复git仓库中指定版本的项目 3）Git分支 一，分支相关命令 1. git branch 查看分支 2. git branch 分支名称 创建分支 3. git checkout 分支名称 切换分支 4. git merge 来源分支 合并分支 5. git branch -d 分支名称 删除分支（分支被合并后才允许删除）（-D 强制删除） 二，暂时保存更改 1. 存储临时改动： git stash 2. 恢复改动： git stash pop 四，注册Github账号 1). 将本地仓库推送到远程仓库 1. git remote add 远程仓库地址别名 远程仓库地址

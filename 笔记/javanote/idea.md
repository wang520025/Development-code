# idea快捷键
- [idea快捷键](https://gitee.com/zhengqingya/idea-config/blob/master/)
- 快速生成main()方法: psvm，回车
- 快速生成输出语句: sout，回车
- 内容提示，代码补全等：Ctrl+Alt+space
- 单行注释：Ctrl+/
- 多行注释：Ctrl+Shift+/
- 格式化：Ctrl+Alt+L
- 查看当前类的所有方法和成员变量的快捷键ctrl+f12和alt+7
- 打开某个类的类型继承图 Ctrl + h
- spring-17-容器，查看某个类的构造方法的快捷键
- 快捷键导包 Alt+Enter
- 生成左边的内容 Ctrl+Alt+V
- alt + 鼠标左键：整列编辑
- 快捷方式导入jar包的坐标：在 pom.xml 中 按 alt + insert，选择 Dependency
